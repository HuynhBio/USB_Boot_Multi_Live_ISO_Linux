Extract DrWeb ISO to drweb folder: e.g.

\ISO_Extract\drweb\casper\vmlinuz (etc.)

Only casper folder is needed.

For persistence, unzip the casper-rw.zip file to the \ISO_Extract\drweb folder  (200MB).

If you want to make a larger persistent file, use RMPrepUSB - Create ext2 FS - casper-rw - casper-rw - (size in MB), and then move the file to the \ISO_Extract\drweb folder.
